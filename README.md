# Window-Registry-Analyzer
Window Registry Analyzer created in Python focusing on user behavior analysis

<br>

## Collect.py
레지스트리 편집기를 열어서 경로 지정 후 추출.

## Collect2.py
레지스트리 명령어 실행을 통해 파일 추출. 

## Collect3.py
특정 폴더에 있는 파일들 중 지정한 파일들 추출.

## Collect4.py
2번과 3번을 합쳐서 전체 파일 추출.

### 관리자 권한으로 실행 할 것.
